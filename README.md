# 啟動

## Rasa Server
```shell
rasa run actions
rasa run
```

## 前台客服端口
```shell
python audio.py
```

## 後台網頁
位於資料夾 `./backend`

### 前端Server
```shell
npm install
npm run start
```
### 後端Server
```shell
python api.py
```
